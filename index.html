¿Qué es HTML?






HTML (HyperText Markup Language) es el lenguaje estándar para crear y estructurar páginas web. Utiliza una serie de elementos o "etiquetas" que indican al navegador cómo debe mostrar el contenido. Estas etiquetas definen la estructura básica de una página web, incluyendo encabezados, párrafos, enlaces, imágenes, listas, formularios y otros elementos.


¿Qué es CSS?






CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación visual de documentos HTML y XML. Con CSS, puedes controlar el diseño, la apariencia y el formato de elementos específicos o de toda una página web.

Estructura Básica de un Documento HTML






Un documento HTML tiene una estructura básica que incluye las etiquetas <html>,<head>, y <body> ;.

Agregar Estilos con CSS






Para aplicar estilos a tu página web de manera organizada y reutilizable, puedes vincular un archivo CSS externo desde tu documento HTML. Esto te permite mantener separados el contenido estructural (HTML) y los estilos visuales (CSS), facilitando la mantenibilidad y la actualización del diseño de tu sitio web.

Ejemplos de estilos basicos

  • Estilo de fondo: Puedes definir un color de fondo para toda la página usando CSS.
  • Estilo de texto: Cambiar el color del texto y ajustar el tamaño de la fuente para que se vea más grande o más pequeño.
  • Estilo de encabezado: Aumentar el tamaño del texto para que los encabezados sean más prominentes o cambiar su color para hacerlos resaltar.
